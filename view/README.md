# aqui ficara todos os arquivos HTML+CSS+JS

## Como Executar

1. Certifique-se de ter o Node.js instalado (https://nodejs.org/).
2. No terminal, navegue até a pasta do projeto view.
3. Execute o comando:

```bash
npm install
```

1. Em seguida, inicie o servidor com:

```bash
npm run start
```
